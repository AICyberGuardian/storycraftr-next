## Recommended Usage Pattern

1. Planning first:
- Use `planner.agent.md` or `implementation-plan.agent.md`.
- For architecture-level reviews, use `se-system-architecture-reviewer.agent.md`.

2. Implementation:
- Use `blueprint-mode-codex.agent.md` for strict execution.
- Use `context-architect.agent.md` for multi-file dependency-aware changes.

3. Quality and safety:
- Use `gem-reviewer.agent.md` and `agent-governance-reviewer.agent.md` for security/governance checks.
- Use `polyglot-test-agent/` and `pytest-coverage/` for tests and coverage hardening.

4. Documentation:
- Use `documentation-writer.agent.md`, `gem-documentation-writer.agent.md`, and `architecture-blueprint-generator/`.
